# БЕКТЕЙЛФАН

## Как запустить
Просто открой `index.html` в браузере (двойной клик). Если звук не будет
играть с первого экрана — это нормально, браузер блокирует автозвук до
клика/нажатия клавиши, поэтому есть экран "НАЖМИ ЧТОБЫ НАЧАТЬ".

Для игры с телефона по Wi-Fi (или чтобы наверняка не было проблем со звуком)
удобнее поднять локальный сервер в папке проекта:
```
python3 -m http.server 8000
```
и открыть `http://localhost:8000` (или `http://IP-компьютера:8000` с телефона).

## Управление
- **ПК**: стрелки — навигация и движение души, Z (или Enter) — подтвердить/ударить, X (или Esc) — назад.
- **Телефон**: на экране появляются свои кнопки (стрелки + OK/НАЗАД) — определяется
  автоматически по типу устройства, на ПК их не будет.

## Структура
- `index.html` — разметка, стили, мобильные кнопки
- `game.js` — вся логика. Сверху два объекта, которые проще всего редактировать:
  - `SETTINGS` — скорости, размеры спрайтов
  - `STORY` — реплики, порядок атак, предметы, ХП
- `assets/` — твои спрайты и звуки

## Сюжет и атаки — как сейчас работает
1. Аксий говорит вступительные реплики (в облачке у головы).
2. Сразу атака №1 — кольцо яблок, которое быстро схлопывается. Музыки ещё нет.
3. Дальше — меню (FIGHT/ACT/ITEM/MERCY), начинает играть `battle_music.mp3` по кругу.
   Показывается текст "Аксий расслаблен."
4. Если выбрать ACT — после текста действия Аксий говорит "Мало..." и идёт атака.
   Если выбрать FIGHT или ITEM — атака начинается сразу, без реплики Аксия.
5. Атака №2 — яблоки летят в случайных направлениях.
6. Атака №3 — две бутылки-бластера: 5 сек прицеливаются (поворачиваются на душу),
   3 сек стреляют прямой струёй.
7. Дальше атаки №2 и №3 чередуются по кругу — можно дописать ещё атак
   в `STORY.attackCycle` и в объект `ATTACKS`.

## Если что-то выглядит не так
- **Бластер целится не той стороной** — открой `game.js`, найди
  `SETTINGS.blasterRotationOffset` и поменяй `Math.PI/2` на `-Math.PI/2`.
- **Хочешь другой размер спрайтов** — поменяй `aksiySpriteHeight`,
  `blasterSpriteHeight`, `appleSize` в `SETTINGS`.
- **"Задонатить"** — текста для этого варианта ACT в описании не было, я
  написал заглушку ("Ты закинул донат Аксию...") — легко поменять в `STORY.actOptions`.
